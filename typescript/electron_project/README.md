# Electron Template

Electron TypeScript project template.

## Prerequisites

- Node.js
- pnpm

## Setup

```bash
pnpm install
```

## Scripts

| Command         | Description                                                  |
| --------------- | ------------------------------------------------------------ |
| `pnpm dev`      | Build and start Electron in development mode                 |
| `pnpm build`    | Full build: compile, bundle renderer, copy assets, obfuscate |
| `pnpm start`    | Build and start Electron                                     |
| `pnpm test`     | Run tests                                                    |
| `pnpm validate` | Format check + lint + type-check                             |
| `pnpm fix`      | Format and lint fix                                          |
| `pnpm pack`     | Create unpacked application bundle                           |
| `pnpm dist`     | Create production installers                                 |
