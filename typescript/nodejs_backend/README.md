# Node.js Template

TypeScript Node.js project template.

## Prerequisites

- Node.js
- pnpm

## Setup

```bash
pnpm install
```

## Scripts

| Command         | Description                      |
| --------------- | -------------------------------- |
| `pnpm dev`      | Start dev server with hot-reload |
| `pnpm build`    | Compile TypeScript to JavaScript |
| `pnpm start`    | Start production server          |
| `pnpm test`     | Run tests                        |
| `pnpm validate` | Type-check + lint + format check |
